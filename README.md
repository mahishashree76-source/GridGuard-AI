# GridGuard-AI ⚡

A Smart Grid Monitoring System that demonstrates fault detection and self-healing in a power distribution network.

## Features

- Fault Detection
- Alternative Route Identification
- Self-Healing Grid Concept
- Network Visualization using NetworkX
- Interactive Dashboard using Streamlit

## Technologies Used

- Python
- Streamlit
- NetworkX
- Matplotlib

## Project Structure

app.py - Streamlit Dashboard

grid.py - Smart Grid Simulation

## How to Run

Install dependencies:

```bash
pip install streamlit networkx matplotlib
```

Run the dashboard:

```bash
streamlit run app.py
```

## Demo

When a fault occurs between substations, the system automatically finds an alternative route and restores connectivity.
